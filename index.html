Logitech G35 Headset - Linux Support Library

This library provides support to handle the Logitech G35 Headset buttons. The library is required by g35d the user-space daemon. The library provides the necessary functions to claim and control the G35 Headset HID and will send the pressed buttons to the uinput device. There are three macro buttons (G1, G2, and G3) and a microphone event (changing the arm position or pressing the mute button) to assign a button press event to.

Requirements

  • uinput
  • cmake (>= 2.8)

Getting the code

Either download one of the archive formats from the top, or clone the git repository: 
$ git clone https://github.com/jaypikay/libg35.git

Building

To compile the project change into the source directory and execute: 
$ cmake .
$ make
If everything went smoothly the application can be installed with (Default Prefix: /usr/local): 
$ make install
